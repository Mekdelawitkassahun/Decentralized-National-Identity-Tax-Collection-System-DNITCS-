# Decentralized-National-Identity-Tax-Collection-System-DNITCS-

Live Demo 
https://monumental-cajeta-678fcf.netlify.app/


Developed a blockchain-based decentralized national identity and tax collection system (DNITCS) using Ethereum smart contracts, React, and Python Flask, enabling role-based portals for citizens, employers, tax collectors, and government admins with automated tax calculations and Fayda ID verification.
